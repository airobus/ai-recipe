# AI 菜谱助手

## 项目描述
一个基于 AI 的菜谱助手，帮助选择困难的用户轻松找到适合的菜谱。

## 功能特点
- AI 推荐菜谱
- 详细菜谱信息
- 制作流程指导
- 菜品制作记录
- 示例图片展示

## 技术栈
- Astro
- TypeScript
- pnpm

## 安装与运行
1. 克隆仓库
2. 运行 `pnpm install`
3. 运行 `pnpm dev`
